# Active Directory Lab

This repsoistory contains hands-on Active Directory labs on Windows Server 2025 built on virtual machines using VMware Workstation

## Skills Demonstrated

- Windows Server 2025 Set Up
- Installation of Active Directory Domain Services
- Assigning server as domain controller
- Joining Windows 11 client to domain controller
- Creation of organizational units (OU's) and Users within Active Directory

## Tools Used

- VMware Workstation
- Windows Server 2025
- Windows 11
- Active Directory
